# HomeworkSnap Marketing Site

Landing site for promoting HomeworkSnap and directing users to the app.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Create a new GitHub repo for this folder.
2. Push code to `main`.
3. Import the repo in Vercel.
4. Deploy with defaults (no env vars required).
